<h3 align="left"><a href="https://github.com/SmokeSculpter/WheresMyTools">Where's My Tools?!</a></h3>

<p style="font-size: 15px;">Where's My Tools?! Is an equipment inventory management system designed to track tools across a worksite. Built with a React front end and an ASP.NET/SQL back end. In this application the user can:</p>

- <p style="font-size: 15px;">View the status of all equipment in inventory</p>
- <p style="font-size: 15px;">Check out equipment out under a specific employee</p>
- <p style="font-size: 15px;">View all currently checked out equipment grouped by employee</p>
- <p style="font-size: 15px;">Check in equipment</p>
- <p style="font-size: 15px;">View the check out history of all equipment</p>

<p style="font-size: 15px;">Detailed documentation of this project as well as what I learned from it can be found <strong><a href="https://github.com/SmokeSculpter/WheresMyTools">here!</a></strong></p>
